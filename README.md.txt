# Student Performance Data Analysis 🎓

This project analyzes a dataset of student performance from Kaggle using Python (Pandas, NumPy, Matplotlib).

## 📊 Features

- Calculates average score per student
- Classifies Pass/Fail based on score ≥ 50
- Analyzes performance based on:
  - Gender
  - Parental education
  - Test preparation course
- Includes visualizations using Matplotlib
- Saves insights and cleaned data to CSV

## 📁 Files

- `student_analysis.ipynb` – Jupyter notebook with full code
- `full_cleaned_data.csv` – Dataset with average score and result
- `gender_pass_fail_analysis.csv` – Pass/Fail by gender
- `parent_education_analysis.csv` – Pass/Fail by parental education
- `test_prep_pass_fail_analysis.csv` – Pass/Fail by test prep

## 📌 Tools Used

- Python 3.x
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## 🔗 Dataset Source

[Kaggle - Student Performance Data](https://www.kaggle.com/datasets/spscientist/students-performance-in-exams)
